# dubbox_demo

#### 介绍
dubbox_demo是一个使用dubbox的demo项目，仅供学习练手使用，访问地址：<http://localhost:8082/user/showName.do> ，如果得到相应结果，即说明服务启动正常。

#### 软件架构
软件架构说明：

使用zookeeper作为注册中心，dubbox作为RPC框架


#### 安装教程

1.  安装虚拟机centos6.5
2.  配置zookeeper注册中心
3.  启动dubbox的RPC服务框架

#### 使用说明
